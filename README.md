# README - PauseMagic
    Скрипт позволяет временно приостановить (suspend) процесс, владеющий активным окном, и все его дочерние процессы при удержании горячих клавиш. 
    После отпускания комбинации процессы автоматически возобновляются (resume). Есть отдельная комбинация для выхода из программы.

# Требования
    Windows.
    Python 3.13.2 (64-bit).
    Пакеты:
        psutil (рекомендуется v7.2.2)
        keyboard (рекомендуется v0.13.5)
    (Для приостановки некоторых процессов могут потребоваться права администратора).

# Файлы
    main.py — основной скрипт (тот код, который вы запускаете).
    keys_bind.json — файл конфигурации горячих клавиш (скрипт создаст его автоматически, если файла нет или он повреждён).
    dump — файл логов/ошибок (скрипт записывает сообщения в этот файл).

# Конфигурация keys_bind.json
    Пример содержимого:
    {
      "keys_bind": "alt + ctrl",
      "keys_exit": "ctrl + shift"
    }

    keys_bind — комбинация, при удержании которой выполняется приостановка процессов.
    keys_exit — комбинация для немедленного выхода из программы.

# Как пользоваться
    Запустите скрипт или exe файл
    Переключитесь на окно процесса, который хотите временно приостановить.
    Нажмите комбинацию из keys_bind (по умолчанию Alt + Ctrl). Скрипт приостановит процесс активного окна и всех его дочерних процессов.
    Прожмите комбинацию еще раз — процессы будут возобновлены.
    Чтобы завершить работу скрипта, нажмите комбинацию keys_exit (по умолчанию Ctrl + Shift).

# Примечания и рекомендации
    Запускайте от имени администратора, если нужно приостанавливать процессы, требующие повышенных прав.
    Не приостанавливайте критичные системные процессы — это может привести к нестабильности системы.
    Если при работе скрипта появляются ошибки, смотрите файл dump для подробностей(зачастую они появляються там).

# Безопасность
    Скрипт не отправляет данные в сеть; логирование происходит локально в файле dump.
